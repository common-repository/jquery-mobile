This plugin install the jQuery Mobile framework on your blog.

== Description ==
This plugin installs the jQuery Mobile framework, which is based on jQuery and jQuery UI.

It`s recommended to use this plugin together with my jQuery Updater plugin for jQuery compatibility. 

This plugin doesn`t turn your site mobile!

== Installation ==
1. Upload `jquery-mobile/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
